How to write Mandarin

Strokes

    The Chinese system has been in continuous use for over 3000 years, with the traditional characters essentially unchanged since the 7th century. There is a story in every character so it is important to learn the correct order of the strokes and even the name of them.

    I have heard Chinese children speak the name of the strokes out loud when they write and to know this can help with remembering how to write them by hand and more importantly, to know the rules, THE ORDER and direction of the strokes when you are writing this beautiful system on paper. It is not language learning without actually writing on paper.

PINYIN Keyboard

    One wonderful way to memorize new characters y to start writing them from day one. This includes knowing how to use the Chinese keyboard. There are many ways to write Chinese but the most popular one is the PINYIN keyboard that is based on our normal keyboard layout (the QWERTY) and the use of the numbers of the keyboard to select the character desired. With Mandarina you will start learning how to use it from day one.
